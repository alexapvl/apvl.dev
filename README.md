# apvl.dev

my personal corner of the internet.

![apvl.dev](https://apvl.dev/og-image.jpg)

## stack

- [Astro](https://astro.build) - static site generator
- [TypeScript](https://www.typescriptlang.org) - type safety
- [Three.js](https://threejs.org) - WebGL background
- [MDX](https://mdxjs.com) - content

## features

- animated noise background that reacts to music
- keyboard navigation (`n` for menu)
- theme toggle (light/dark)
- music player

## running locally

```bash
pnpm install
pnpm dev
```
